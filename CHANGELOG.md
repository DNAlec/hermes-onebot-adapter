# Changelog

## [1.0.0b1] - 2026-07-02

First public beta release.

- OneBot 11 reverse/forward WS transport
- Hermes plugin bridge with toolset management
- WebUI dashboard, connection management, command filtering
- Per-group config with member filtering
- Slash-command permission model (everyone/admin/disabled)
- 28 OneBot API tools exposed to LLM
- Voice transcoding via ffmpeg
- SeqMap for NapCat real_seq ↔ message_id mapping
